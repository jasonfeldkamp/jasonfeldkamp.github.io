Missile Tracking

A script that finds the best trajectory for a friendly missile to intercept an enemy missile.

Download Code [.zip]

While surfing Wikipedia, I found that homing missiles have a guidance system that either uses infrared or radar sensors to determine the distance away from enemy objects. I was then inspired to write a MATLAB script that would take this distance data and find the best intercept trajectory for a counter-missile.

The friendly missile first observes the enemy missile and collects 3 position values over 3 time steps. It then numerically differentiates these values to get 2 velocity values and 1 acceleration value. The algorithm then predicts the enemy missile’s flight path using Euler’s method, including the time, position, velocity and acceleration at each point along the path. Now the fun begins. The algorithm calculates the time it would take for the friendly missile to intercept the enemy missile at each point along its predicted flight path, taking into consideration the friendly missile’s current position, velocity, and acceleration. Next, the error between the times of the predicted enemy missile and the friendly missile intercept are calculated at each point along the path. The minimum of this error function corresponds to the index of the point along the flight path that would be the best trajectory, and the missile points in that direction. This entire process is repeated for each time step until intercept occurs. This algorithm will work for an enemy missile undergoing any flight path or acceleration, not just gravity.

The 1st clip is with low range tracking and the rest are with a much higher trajectory prediction range. Also the animations of the side by side graph shows the iterative minimization of the error function until impact occurs.
